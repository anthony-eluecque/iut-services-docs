## Commandes

* `mkdocs new [dir-name]` - Créer un nouveau projet.
* `mkdocs serve` - Live-reloading la documentation
* `mkdocs build` - Build la documentation web
* `mkdocs -h` - Commande d'aide (Help)

## Structure du projet
